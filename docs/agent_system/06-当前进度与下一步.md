## 下一步: Phase 2 — 四层记忆与 LLM 决策

### 目标

Agent 有完整的四层记忆系统，LLM 基于 Life Pattern + 记忆生成日计划。

### 新建文件

```
synthetic_socio_wind_tunnel/agent/memory.py    # ~250-300 行
```

### 要实现的内容

**1. 数据模型**

```python
class LifePatternEntry(BaseModel):
    """Layer 4: 一条生活模式记录。"""
    pattern_id: str
    category: str           # "work" / "commute" / "dining" / "leisure" / "social" / "shopping"
    description: str        # "我在 TechCorp 上班，周一到周五 9:00-18:00"
    locations: list[str]    # 相关的 location_id
    schedule: str           # "weekdays 9:00-18:00" / "saturday morning"
    confidence: float       # 0-1, 模式稳定度
    formed_at: datetime
    last_confirmed: datetime
    source: str             # "initial_setup" / "behavior_learned" / "event_triggered"

class MemoryEntry(BaseModel):
    """Layer 1-3: 单条记忆。"""
    memory_id: str
    timestamp: datetime
    memory_type: str        # "event" / "summary" / "reflection"
    content: str
    location_id: str | None
    related_agents: list[str]
    importance: float       # 0-1
    emotional_valence: float  # -1 到 +1
```

**2. AgentMemory 类**

```python
class AgentMemory:
    # Layer 4 操作
    get_daily_framework(day_of_week) → list[LifePatternEntry]
    initialize_from_profile(profile, llm_client)  # 从 Profile 生成初始 Life Pattern
    update_patterns(llm_client, recent_summaries)  # 根据近期行为更新模式

    # Layer 1-3 操作
    record_event(event) → MemoryEntry              # 自动，不需要 LLM
    summarize(llm_client, recent_events) → MemoryEntry  # LLM 压缩
    reflect(llm_client) → MemoryEntry               # LLM 归纳
    retrieve(query, k=5) → list[MemoryEntry]         # 基于规则的检索（不用 embedding）
```

**3. 检索策略（不用 embedding）**

- 时间衰减（指数，半衰期 3 天）
- 重要性加权
- 关键词匹配
- 类型优先级：reflection > summary > event
- 相关 agent 匹配

**4. 更新 Planner**

- `generate_daily_plan()` 增加 `memory: AgentMemory` 参数
- Protagonist 的 prompt 追加反思和附近信息
- 高配 agent 的 prompt 追加近期记忆

### 验证方式

```
创建 1 个 Protagonist agent
→ 从 Profile 初始化 Life Pattern (Layer 4)
→ LLM 基于 Life Pattern 生成日计划
→ 移动过程中记录 Layer 1 事件
→ 日终生成 Layer 2 摘要和 Layer 3 反思
→ 第二天的计划参考昨天的记忆
→ 连续跑 3 天后检查 Life Pattern 是否被行为更新
```
